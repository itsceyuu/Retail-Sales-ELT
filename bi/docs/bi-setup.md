# BI Environment Setup

Author: Dzikri

## 1. Overview
Dokumen ini berisi langkah awal setup environment untuk kebutuhan Business Intelligence (BI) dalam proyek Retail Sales Analytics.

## 2. Setup yang Telah Dilakukan
- Clone repository dari GitHub
- Setup Visual Studio Code
- Membuat branch khusus BI (bi/dzikri-dashboard)
- Menyiapkan struktur folder BI

## 3. Struktur Folder BI
- bi/queries → menyimpan SQL query
- bi/dashboard_design → desain dashboard
- bi/docs → dokumentasi BI

## 4. Tujuan
Menyiapkan environment yang terstruktur agar pengembangan dashboard BI dapat berjalan dengan efektif dan kolaboratif.
